FIBA rules with modified overtime.

### Overtime Rules
- **Duration:** 7 minutes stop time
- **Win Condition:** First to 7 points
- **Regular Season:** Game recorded as tie if still tied after OT
- **Playoffs:** Sudden death if still tied
